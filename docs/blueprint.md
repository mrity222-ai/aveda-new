# **App Name**: Aveda Digital

## Core Features:

- Informational Home Page: Display key information like headline, services preview, results, and a clear call to action.
- Comprehensive Services Page: Showcase Aveda's range of services (Web Development, App Development, UI/UX Design, etc.) with descriptions and potential integrations with a portfolio.
- Interactive Contact Form: Implement a contact form that captures user information (Name, Email, Phone, Service Required, Message) and sends it to Aveda Technologies.
- Portfolio Showcase: Showcase past projects with details on client goals, solutions implemented, technologies used, and results achieved. Sortable by industry and service type.
- AI-Powered Insights Generation: Generates summaries or key takeaways from blog posts, saving users time, or suggesting related articles. LLM will act as a tool deciding whether a particular fact is crucial to be included in the summarization.
- Results Tracking: Display statistics such as projects delivered, client satisfaction, industries served, and years of experience.

## Style Guidelines:

- Primary color: Vibrant red (#E53935) to reflect the brand's energy and premium feel, aligning with the provided color scheme.
- Background color: Light gray (#F5F5F5), a desaturated version of the red, provides a clean, modern backdrop.
- Accent color: Orange (#EF6C00), an analogous color to red, for calls to action and highlighting key elements, enhancing the results-driven tone.
- Headline font: 'Poppins', a geometric sans-serif to communicate a precise, contemporary, fashionable, avant-garde look.
- Body font: 'Inter', a grotesque-style sans-serif providing a modern, objective, neutral look.
- Use modern, minimalist icons in red and black to complement the color scheme.
- Implement a clean, grid-based layout for easy navigation and clear information hierarchy.
